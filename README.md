# 🐹 EBMouse — EBM 鼠出任務

**你的實證醫學小幫手 | Your EBM Learning Companion**

EBMouse is a bilingual (繁體中文 / English) guided learning platform that helps healthcare professionals learn and practice evidence-based medicine through real clinical scenarios.

## What is EBMouse?

Follow a friendly hamster mascot through the complete EBM workflow — from reading a clinical scenario to forming a clinical recommendation. Each step teaches you the methodology while you build your own EBM project.

### The 5A Workflow

| Step | What You Learn | What You Produce |
|------|---------------|-----------------|
| **Assess** 🔍 | How to analyze a clinical scenario | Patient profile & background summary |
| **Ask** ❓ | How to formulate a good clinical question | Structured PICOT |
| **Acquire** 📚 | How to search and screen literature | Search strategy & article selection |
| **Appraise** ⚖️ | How to critically evaluate evidence | CASP appraisal & GRADE assessment |
| **Apply** 💊 | How to translate evidence into practice | Clinical recommendation |

## Who Is This For?

- Pharmacists and healthcare professionals preparing for EBM competitions
- Junior clinicians learning evidence-based practice
- Anyone who wants a structured, guided approach to EBM

## Features

- 🌐 Bilingual interface (繁體中文 / English)
- 🐹 Friendly hamster mascot guides you through each step
- 📖 27 inline teaching blocks covering the full 5A methodology
- 🧰 8 interactive Toolbox reference guides (PICOT, CASP, Core GRADE, MID, EtD, PubMed Search, SRA, LitSuggest)
- 🏋️ 72 practice questions across PICOT, CASP, GRADE, and EtD (MC, fill-in-blank, error-spotting)
- 📚 3 fully annotated Case Library entries with real competition data
- 💾 Auto-save with JSON export/import
- 📎 Export your work as a presentation
